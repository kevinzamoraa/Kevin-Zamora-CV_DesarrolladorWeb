La directiva *ngSwitch es una directiva estructural en Angular que se utiliza para realizar una selección condicional entre varias opciones. Proporciona una forma de alternar entre diferentes bloques de contenido basándose en el valor de una expresión.
Se utiliza en combinación con las directivas *ngSwitchCase y *ngSwitchDefault
